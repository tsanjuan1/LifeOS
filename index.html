02

Architecture

Browser / PWA
Next.js App Router · TanStack Query · shadcn/ui · Recharts · Zustand (UI only)
↕ HTTPS
API Layer — Next.js Route Handlers /api/v1/*
Zod validation · Drizzle ORM · Supabase Auth middleware · File parsing (CSV) · Business logic
↕ Postgres wire protocol
Supabase (Managed)
PostgreSQL + RLS · Auth (JWT) · Storage (CSV/PDF) · pg_cron (scheduled jobs) · Edge Functions (async tasks)
RLS Security
Every table has Row Level Security. All queries auto-scoped to auth.uid(). No manual filtering needed.
Monolith First
Single Next.js repo. Clean module boundaries in lib/. Extract microservices only at Phase 3.
Hybrid Rendering
Server Components for layout/auth. Client Components for interactive widgets. RSC streaming for dashboard.

Personal Score Formula

30%
Finance
savings_rate ≥ 20%: +40
no overspend: +30
subs reviewed: +15
no anomalies: +15
30%
Productivity
completion_rate × 60
focus done: +20
no overdue: +20
-5 per overdue task
25%
Habits
(done/scheduled) × 70
streak bonus: min(30,
longest_streak × 2)
15%
Goals
avg(progress%) active
on-track bonus: +20
if ahead of schedule
finalScore = finance × 0.30 + productivity × 0.30 + habits × 0.25 + goals × 0.15 // Computed nightly via pg_cron → stored in personal_scores

Quick Start

  • 1
    Scaffold Next.js project
    npx create-next-app@latest lifeos --typescript --tailwind --app --src-dir
  • 2
    Install dependencies
    npm install drizzle-orm @supabase/supabase-js @supabase/ssr @tanstack/react-query react-hook-form zod recharts papaparse && npm install drizzle-kit --save-dev && npx shadcn@latest init
  • 3
    Create Supabase project → copy credentials to .env.local
    NEXT_PUBLIC_SUPABASE_URL=... · NEXT_PUBLIC_SUPABASE_ANON_KEY=... · DATABASE_URL=postgresql://...
  • 4
    Run schema migration in Supabase SQL editor
    Paste the full schema SQL from lifeos-architecture.md → Run
  • 5
    Enable RLS policies
    Run the RLS policy SQL block for all 14 tables
  • 6
    Connect GitHub → Vercel for auto-deploy
    vercel.com → Import Git Repository → Add env vars → Deploy
  • 7
    Start building
    npm run dev → localhost:3000
